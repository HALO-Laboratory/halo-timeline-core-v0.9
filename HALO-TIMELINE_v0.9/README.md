# HALO-TIMELINE v0.9 (Timeline Core)

> まず一度動かします。理解はそこから始まります。  
> *(First, get it to run once. Understanding starts from there.)*

This repo contains a single PowerShell script that merges CSVs for EVTX / LNK / JumpList / MFT / USNJ into one timeline.
It aims to keep things minimal and reproducible for learners.

See `docs/` for details and `examples/` for a tiny synthetic example.
