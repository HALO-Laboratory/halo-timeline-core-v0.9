# HALO-TIMELINE v0.9 Release Notes

## Purpose
v0.9 は、タイムライン統合部分に焦点を当てた **Core Edition** です。

## Why a Core Edition?
複数ツールや環境設定を含む構成は、初回実行においてつまずく要素が多くなります。  
本版は、以下を明確にするために **最小構成** としています：

- 入力 → 正規化 → 統合という処理の本質
- UTC 時刻への統一と列スキーマ整理
- 再現性と理解の優先

## Value
- 動作が追いやすい
- エラー切り分けが容易
- 学習が進むほど拡張点が見える

## Future
v1.x 系では、アーティファクト抽出・ワークフロー化・自動化統合が段階的に復活します。  
v0.9 は、そのための **基準点** です。
